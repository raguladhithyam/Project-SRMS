STEPS:
1) Install MongoDB version 4.0.28 from https://www.mongodb.com/try/download/community
(Skip Step 1 if already installed)
2) Open CMD type md \data\db
3) Type "C:\Program Files\MongoDB\Server\4.0\bin\mongod.exe"
4) Open Another Command Prompt Window and type "C:\Program Files\MongoDB\Server\4.0\bin\mongo.exe"
5) Shell will open in the type show dbs
6) Open MongoDB Compass and make the connection
7) In MongoDB shell type exit and close that window
8) After the work stop the server by pressing CTRL + C
